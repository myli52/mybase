# 李明洋个人网站

基于 Nuxt 3 + Vue 3 + TypeScript + Tailwind CSS 构建的个人网站。

## 项目特性

- ✨ 现代化的设计和交互体验
- 📱 完整的响应式布局,支持 PC/平板/移动端
- 🎨 使用 Tailwind CSS 进行样式开发
- 📝 支持 Markdown 博客文章
- ⚡️ Nuxt 3 提供的极致性能
- 🔍 SEO 友好

## 技术栈

- **框架**: Nuxt 3
- **UI 库**: Vue 3  
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **博客**: Nuxt Content
- **部署**: Vercel / Cloudflare Pages

## 开发指南

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看网站

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```
